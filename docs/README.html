Steam Multi-Box Launcher

Aplikasi GUI Windows untuk menjalankan beberapa akun Steam milik sendiri secara terisolasi di profil Sandboxie, dengan auto-login bergiliran, verifikasi akun, auto-save Remember Me, retry pintar, dan notifikasi Discord opsional.

Prasyarat

  • Sandboxie (Classic atau Plus) terinstal — unduh dari github.com/sandboxie-plus/Sandboxie. Tool mendeteksi otomatis di drive mana pun.
  • Steam terinstal.
  • Python 3.10+.

Instalasi

pip install -r requirements.txt

Dependensi: customtkinter (GUI), pywinauto (UI automation), pytest (test). Selebihnya pakai stdlib.

Fitur Utama

FiturPenjelasan singkat
Auto-detect Sandboxie & Steam Pakai registry & service path; juga ada tombol Auto Find + Browse manual di Pengaturan.
Run bergiliran Satu akun selesai login, baru lanjut akun berikutnya. Tidak spam.
UI form login + Remember Me Tool isi form login & centang Remember me → Steam simpan encrypted token → manual reopen box auto-login.
Verifikasi akun via VDF Baca loginusers.vdf di box, cocokkan AccountName + MostRecent="1" dengan akun di accounts.txt. Tahu kalau Steam salah akun.
Auto recovery dari mismatch Kalau akun salah login → wipe sesi + retry sampai max_retries+1 kali.
Stop interruptible Tombol Stop benar-benar menghentikan controller thread, tidak menunggu polling habis.
Checkbox per-baris + Run Tercentang Pilih sebagian akun saja untuk di-Run.
Edit Credential per box Tombol ✏️ di tiap baris → ubah username/password → tool wipe sesi lama & login ulang. Box lama auto-delete kalau username berubah.
Hapus box per baris (🗑) atau sidebar Hapus aman: terminate proses → delete_sandbox → edit Sandboxie.ini.
Tutup Steam host otomatis Sebelum Run, tool deteksi steam.exe host yang jalan & minta izin tutup (single-instance Steam menghalangi sandbox).
Discord webhook (opsional) Notif run mulai, tiap akun sukses/gagal, ringkasan akhir, dan edit credential.

Quick Start

Lihat Tata Cara Pakai untuk panduan langkah demi langkah yang detail. Ringkasnya:

  1. Salin data/accounts.example.txt menjadi data/accounts.txt, isi dengan format username,password (atau username|password), satu akun per baris.
  2. Jalankan: python app.py
  3. Klik Pengaturan; pastikan path Sandboxie & Steam terisi (kalau tidak: klik Auto Find atau Browse).
  4. Klik Run Semua. Kalau muncul "Steam host terdeteksi", pilih Yes.
  5. Approve prompt Steam Guard 2FA manual di tiap jendela Steam yang muncul (kalau aktif untuk akun tersebut).
  6. Setelah selesai: buka box manual via SandMan → Steam akan auto-login ke akun yang sesuai box.

Lokasi File

Semua data milik pengguna ada di folder data/ di samping app.py, supaya mudah dicari dan di-backup:

  • data/accounts.txt — daftar kredensial (yang Anda isi).
  • data/accounts.example.txt — contoh format.
  • data/fail.txt — ditulis otomatis tiap Run untuk akun gagal, format username,password  # alasan.
  • config.json — preferensi tool (paths, delays, webhook URL).

Lokasi data/accounts.txt bisa dipindah lewat Pengaturan > Browse; fail.txt mengikuti folder yang sama.

Keamanan & Batasan

⚠️ Penting: data/accounts.txt dan data/fail.txt menyimpan password dalam teks biasa. Batasi izin akses folder data/ (NTFS permission). Folder data/ & file di dalamnya sudah masuk .gitignore.

Batasan tool ini (sengaja):

  • Tidak men-generate kode Steam Guard 2FA. User approve manual via HP/email.
  • Tidak menyimpan shared_secret / maFile.
  • Tidak membuat akun Steam.
  • Hanya untuk akun milik sendiri (keluarga, akun alt pribadi, testing).

Troubleshoot Singkat

GejalaCek dulu
"Sandboxie/Steam belum terdeteksi" Pengaturan → Auto Find, atau Browse manual ke folder Sandboxie (yang berisi Start.exe) dan steam.exe.
Akun "ngikut" Steam host Klik Tutup Steam Host di sidebar sebelum Run.
"Too Many Retries" dari Steam Tunggu 5-30 menit (cooldown rate-limit Steam). Tool sudah punya deteksi rate-limit dan tidak retry membabi-buta.
Box di-create tapi tidak kelihatan di SandMan Tutup & buka kembali SandMan, atau klik View → Refresh.
Manual reopen box minta login lagi Pastikan login_method=ui (default) di config — itu yang centang Remember me & bikin token tersimpan.
Discord Test gagal Pop-up sekarang menampilkan kode HTTP / error eksak. Cek URL webhook (harus diawali https://discord.com/api/webhooks/...).
